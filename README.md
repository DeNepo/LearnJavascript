## 2. SUGGESTED CODING RULES AND BEST PRACTIVES

* Code using "use strict"; 
* Always use semicolons. 
* Prefer using === over == 
* Always use curly braces 
* Avoid/Reduce global variables 
* Avoid using with() 
* Avoid using eval()
* Use JSHint / JSLint
